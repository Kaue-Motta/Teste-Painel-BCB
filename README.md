# Painel de Crédito - BCB
Projeto automatizado com as estatísticas monetárias e de crédito do Banco Central.

O projeto consiste em um painel de quatro páginas que busca trazer visiblidade das informações de crédito divulgadas pelo BC. Cada aba se trata de um indicador diferente com distintas visões, abordando a situação atual, evolução, tendência e riscos do mercado de crédito brasileiro de forma visual e intuitiva.
